# Audio Files

Place your sound files in the respective profile folders:

- `linear/` - Linear switch sounds
- `clicky/` - Clicky switch sounds  
- `typewriter/` - Typewriter sounds

## Required Files

Each profile folder should contain:
- `key.mp3` (or `.wav` or `.ogg`) - Default keypress sound
- `spacebar.mp3` - Spacebar sound
- `enter.mp3` - Enter key sound
- `backspace.mp3` - Backspace key sound

## Format

Supported formats: MP3, WAV, OGG

## Fallback

If audio files are not found, the extension will generate simple fallback sounds programmatically.
